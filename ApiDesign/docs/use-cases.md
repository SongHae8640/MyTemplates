# Subject
링크 큐레이션 시스템(이후 '시스템'으로 통칭) 은 사용자가 웹페이지의 링크를 관리하고, 광고주가 광고를 넣을 수 있는 시스템이다. 

## Glossary
- 사용자(User): 시스템을 사용하는 사람
- 관리자(Admin): 시스템을 관리하는 사람
- 광고주(Advertiser): 시스템에 광고를 넣으려고 하는 사람
- 링크(Link): 웹페이지의 주소
- 광고(Ad) : 사용자에게 노출되는 광고
